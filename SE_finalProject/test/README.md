# Test Review Area

This folder is reserved for team members to upload or commit test-review notes, manual QA results, screenshots, or sample test cases.

Suggested naming:
- `reviews/<name>-<feature>-review.md`
- `reviews/<name>-<feature>-test-cases.md`

Keep test notes separate from production code so reviewers can see verification work clearly in git.
